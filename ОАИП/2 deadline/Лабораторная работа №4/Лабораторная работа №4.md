#### Задание 1
Создайте программу для учета оценок студентов:
1. Создайте пустой список `grades`
2. Добавьте в него 5 произвольных оценок (от 1 до 5) с помощью метода `.append()`
3. Выведите список в формате: `Текущие оценки: [ваш список]`
4. Удалите первую и последнюю оценку из списка
5. Найдите и выведите средний балл студентов
6. Выведите итоговый список оценок

**Пример:**
Вывод:
```
Текущие оценки: [5, 4, 3, 5, 2]
Средний балл: 3.5
Итоговые оценки: [4, 3, 5]
```

```python
grades = []
grades.append(5)
grades.append(4)
grades.append(3)
grades.append(5)
grades.append(2)
print("Текущие оценки:", grades)
del grades[0]
del grades[-1]
summa = 0
for ocenka in grades:
    summa = summa + ocenka
sredniy_ball = summa / len(grades)
print("Средний балл:", sredniy_ball)
print("Итоговые оценки:", grades)
```

#### Задача 3
Создайте программу для работы со списком задач (to-do list):
1. Создайте пустой список `tasks`
2. Реализуйте меню:
   - 1 - Показать все задачи
   - 2 - Добавить задачу
   - 3 - Отметить задачу как выполненную (удалить)
   - 4 - Выйти
3. Функционал:
   - При добавлении задачи запрашивайте ее описание
   - При удалении - показывайте пронумерованный список и запрашивайте номер для удаления
   - При выводе задач показывайте их в формате: `1. [задача]`

**Пример:**
```
Ваши задачи:
1. Сделать домашку
2. Купить молоко
Какую задачу выполнили? 1
Задача "Сделать домашку" удалена!
```

```python
tasks = []
while True:
    print("--- Меню ---")
    print("1 - Показать все задачи")
    print("2 - Добавить задачу")
    print("3 - Отметить задачу как выполненную")
    print("4 - Выйти")
    choice = input("Выберите действие (1-4): ")
    if choice == "1":
        print("--- Ваши задачи ---")
        if len(tasks) == 0:
            print("Задач пока нет!")
        else:
            for i in range(len(tasks)):
                print(f"{i+1}. {tasks[i]}")
    elif choice == "2":
        new_task = input("Введите описание задачи: ")
        tasks.append(new_task)
        print(f'Задача "{new_task}" добавлена!')
    elif choice == "3":
        print("--- Ваши задачи ---")
        if len(tasks) == 0:
            print("Задач пока нет!")
        else:
            for i in range(len(tasks)):
                print(f"{i+1}. {tasks[i]}")
            task_number = input("Какую задачу выполнили? Введите номер: ")
            if task_number.isdigit():
                task_number = int(task_number)
                if task_number >= 1 and task_number <= len(tasks):
                    removed_task = tasks[task_number - 1]
                    del tasks[task_number - 1]
                    print(f'Задача "{removed_task}" удалена!')
                else:
                    print("Неправильный номер задачи!")
            else:
                print("Пожалуйста, введите число!")
    elif choice == "4":
        print("До свидания!")
        break
    else:
        print("Неправильный выбор! Введите число от 1 до 4")
```
#### Задача 4
Проанализируйте данные о температурах:
1. Создайте список среднесуточных температур за 2 недели (14 дней)
2. Найдите и выведите:
   - Самую высокую и самую низкую температуру
   - Среднюю температуру за период
   - Количество дней с температурой выше средней
   - Отсортированный список температур по возрастанию
3. Создайте новый список, преобразовав температуры из Цельсия в Фаренгейты по формуле: `F = C × 9/5 + 32`

**Пример:**
```
Температуры: [15, 18, 12, 20, 16, 14, 19, 17, 13, 21, 15, 16, 18, 20]
Максимальная: 21°C
Минимальная: 12°C
Средняя: 16.4°C
Дней выше средней: 7
```

```python
temperatures = [15, 18, 12, 20, 16, 14, 19, 17, 13, 21, 15, 16, 18, 20]
max_temp = temperatures[0]
for temp in temperatures:
    if temp > max_temp:
        max_temp = temp
min_temp = temperatures[0]
for temp in temperatures:
    if temp < min_temp:
        min_temp = temp
sum_temp = 0
for temp in temperatures:
    sum_temp = sum_temp + temp
average_temp = sum_temp / len(temperatures)
days_above_average = 0
for temp in temperatures:
    if temp > average_temp:
        days_above_average = days_above_average + 1
sorted_temperatures = temperatures.copy()
for i in range(len(sorted_temperatures)):
    for j in range(len(sorted_temperatures) - 1):
        if sorted_temperatures[j] > sorted_temperatures[j + 1]:
            temp_value = sorted_temperatures[j]
            sorted_temperatures[j] = sorted_temperatures[j + 1]
            sorted_temperatures[j + 1] = temp_value
fahrenheit_temperatures = []
for temp in temperatures:
    fahrenheit = temp * 9/5 + 32
    fahrenheit_temperatures.append(fahrenheit)
print("Температуры:", temperatures)
print("Максимальная:", max_temp, "°C")
print("Минимальная:", min_temp, "°C")
print("Средняя:", round(average_temp, 1), "°C")
print("Дней выше средней:", days_above_average)
print("Отсортированные температуры:", sorted_temperatures)
print("Температуры в Фаренгейтах:", [round(f, 1) for f in fahrenheit_temperatures])
```
#### Задача 5
Реализуйте программу для статистического анализа текста:
1. Пользователь вводит произвольный текст (может быть многострочным)
2. Программа должна:
   - Разбить текст на слова (игнорируя знаки препинания)
   - Создать список слов в нижнем регистре
   - Найти самое длинное и самое короткое слово
   - Посчитать среднюю длину слова
   - Вывести 5 самых частых слов

**Пример:**
```
Введите текст: Привет мир! Это тестовый текст. Привет еще раз.

Самое длинное слово: тестовый
Самое короткое слово: мир
Средняя длина: 4.8
Топ-5 слов: [('привет', 2), ('мир', 1), ('это', 1), ('тестовый', 1), ('текст', 1)]
```

```python
text = input("Введите текст: ")

# 2. Очищаем текст от знаков препинания
clean_text = ""
for char in text:
    if char.isalpha() or char == " ":
        clean_text = clean_text + char
    else:
        clean_text = clean_text + " "

# 3. Разбиваем текст на слова и переводим в нижний регистр
words = clean_text.split()
lower_words = []
for word in words:
    lower_words.append(word.lower())

# 4. Находим самое длинное слово
longest_word = ""
for word in lower_words:
    if len(word) > len(longest_word):
        longest_word = word

# 5. Находим самое короткое слово (исключая пустые строки)
shortest_word = lower_words[0]
for word in lower_words:
    if len(word) < len(shortest_word) and len(word) > 0:
        shortest_word = word

# 6. Считаем среднюю длину слова
total_length = 0
for word in lower_words:
    total_length = total_length + len(word)

average_length = total_length / len(lower_words)

# 7. Считаем частоту слов
word_count = {}
for word in lower_words:
    if word in word_count:
        word_count[word] = word_count[word] + 1
    else:
        word_count[word] = 1

# 8. Находим 5 самых частых слов
# Создаем список из пар (слово, количество)
pairs = []
for word, count in word_count.items():
    pairs.append((word, count))

# Сортируем по количеству (от большего к меньшему)
for i in range(len(pairs)):
    for j in range(len(pairs) - 1):
        if pairs[j][1] < pairs[j + 1][1]:
            # Меняем местами
            temp = pairs[j]
            pairs[j] = pairs[j + 1]
            pairs[j + 1] = temp

# Берем первые 5 элементов
top_words = pairs[:5]

# Выводим результаты
print("\nРезультаты анализа:")
print("Самое длинное слово:", longest_word)
print("Самое короткое слово:", shortest_word)
print("Средняя длина:", round(average_length, 1))
print("Топ-5 слов:", top_words)
```
