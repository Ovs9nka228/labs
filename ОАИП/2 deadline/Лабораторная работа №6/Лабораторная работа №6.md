#### Задание 1
Напишите функцию `greet_user`, которая принимает имя пользователя в качестве аргумента и выводит на экран персональное приветствие.

**Пример:**
Вызов функции: `greet_user("Алиса")`

```python
def greet_user(name):

greet_user("Алиса")
greet_user("Петя")
greet_user("Мария")
```
#### Задание 2
Создайте функцию `simple_calculator`, которая принимает три аргумента: два числа и строку с оператором (`'+'`, `'-'`, `'*'`, `'/'`). Функция должна возвращать результат математической операции.

**Пример:**
Вызов функции:
```python
result = simple_calculator(10, 5, '*')
print(result)
```

Вывод: 
```text
50
```

```python
def simple_calculator(num1, num2, operator):
    if operator == "+":
        result = num1 + num2
    elif operator == "-":
        result = num1 - num2
    elif operator == "*":
        result = num1 * num2
    elif operator == "/":
        result = num1 / num2
    else:
        result = "Ошибка: неизвестный оператор"
    return result
result1 = simple_calculator(10, 5, "*")
print(result1)
result2 = simple_calculator(10, 5, "+")
print(result2)
result3 = simple_calculator(10, 5, "-")
print(result3)
result4 = simple_calculator(10, 5, "/")
print(result4)
```
#### Задание 3 
Напишите функцию `power`, которая принимает два аргумента: число и степень. Функция должна возводить число в указанную степень. Если степень не передана, функция должна по умолчанию возводить число в квадрат (степень 2).

**Пример:** 
Вызов функции: 
```python 
print(power(3, 3)) 
print(power(5))
```

Вывод:
```text
27
25
```

```python
def power(number, exponent=2):
    result = 1
    for i in range(exponent):
        result = result * number
    return result
print(power(3, 3))
print(power(5))
```
#### Задание 4
Напишите функцию `find_max`, которая принимает два числа и возвращает большее из них.

**Пример:**
Вызов функции: 
```python
max_num = find_max(101, 25)
print(max_num)
```

Вывод:
```text
101
```

```python
def find_max(a, b):
    if a > b:
        return a
    else:
        return b
max_num = find_max(101, 25)
print(max_num)
```
#### Задание 5
Создайте функцию `display_user_profile`, которая принимает в качестве аргументов имя, возраст и город пользователя. Функция должна выводить информацию в отформатированном виде. Продемонстрируйте вызов функции с использованием именованных аргументов, передав их не по порядку.

**Пример:**
Вызов функции:
```python
display_user_profile(age=28, name="Иван", city="Москва")
```

Вывод:
```
Имя: Иван
Возраст: 28
Город: Москва
```

```python
def display_user_profile(name, age, city):
    print("Имя:", name)
    print("Возраст:", age)
    print("Город:", city)
    print()
display_user_profile(age=28, name="Иван", city="Москва")
display_user_profile(city="Санкт-Петербург", name="Мария", age=25)
display_user_profile(name="Петр", city="Казань", age=30)
```
