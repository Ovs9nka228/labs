#### Задание 1
Напишите функцию `sum_numbers`, которая принимает произвольное количество числовых аргументов и возвращает их сумму. ==Добавьте к функции аннотации типов и строку документации (docstring)==.

**Пример:**
Вызов функции: 
```python
print(sum_numbers(1, 2, 3, 4, 5))
```

Вывод:
```text
15
```

*Примечание: Для решения используйте `*args`.*

```python
def sum_numbers(*args):
    total = 0
    for number in args:
        total = total + number
    return total
print(sum_numbers(1, 2, 3, 4, 5))
print(sum_numbers(10, 20))
print(sum_numbers(1.5, 2.5, 3.5))
```
#### Задание 2
Создайте функцию `build_user_profile`, которая принимает обязательный аргумент `user_id` и произвольное количество именованных аргументов с дополнительной информацией о пользователе (например, `name="Иван"`, `city="Томск"`). ==Функция должна возвращать словарь, в котором `user_id` и вся дополнительная информация объединены.==

**Пример:**
Вызов функции:
```python
profile = build_user_profile(101, name="Анна", status="online", email="anna@example.com")

print(profile)
```

Вывод:
```text
{'user_id': 101, 'name': 'Анна', 'status': 'online', 'email': 'anna@example.com'}
```

*Примечание: Используйте `**kwargs` для сбора именованных аргументов.*

```python
def build_user_profile(user_id, **kwargs):
    profile = {}
    profile["user_id"] = user_id
    for key, value in kwargs.items():
        profile[key] = value
    return profile
profile = build_user_profile(101, name="Анна", status="online", email="anna@example.com")
print(profile)

```

#### Задание 4
Создайте глобальную переменную `call_counter`, равную 0. Напишите функцию `increment_counter`, которая при каждом вызове увеличивает эту переменную на 1. ==Функция не должна ничего возвращать==.

**Пример:**
Вызов функции:
```python
increment_counter()
increment_counter()
increment_counter()
print(call_counter)

```

Вывод:
```text
3
```

*Примечание: Используйте ключевое слово `global` для изменения переменной.*

```python
call_counter = 0
def increment_counter():
    global call_counter
    call_counter = call_counter + 1
print("До вызовов функции:", call_counter)
increment_counter()
print("После первого вызова:", call_counter)
increment_counter()
print("После второго вызова:", call_counter)
increment_counter()
print("После третьего вызова:", call_counter)
```
#### Задание 5
Напишите функцию `format_report`, которая принимает один обязательный аргумент `report_title` (название отчета), произвольное количество позиционных аргументов `*data` (пункты отчета) и произвольное количество именованных аргументов `**properties` (метаданные, например, `author="Петров"`). ==Функция должна выводить отформатированный отчет.==

**Пример:**
Вызов функции:
```python
format_report(
    "Ежедневный отчет",
    "Продажи выросли на 10%",
    "Новых клиентов: 5",
    author="Сидоров А.В.",
    date="2025-11-04"
)
```

Вывод:
```text
--- Отчет: Ежедневный отчет ---
Данные:
 - Продажи выросли на 10%
 - Новых клиентов: 5

Свойства:
 - author: Сидоров А.В.
 - date: 2025-11-04
------------------------------
```

```python
def format_report(report_title, *data, **properties):
    print(f"--- Отчет: {report_title} ---")
    print()
    print("Данные:")
    if len(data) == 0:
        print("- Данных нет")
    else:
        for item in data:
            print(f"- {item}")
    print()
    print("Свойства:")
    if len(properties) == 0:
        print("- Свойств нет")
    else:
        for key, value in properties.items():
            print(f"- {key}: {value}")
    print()
format_report(
    "Ежедневный отчет",
    "Продажи выросли на 10%",
    "Новых клиентов: 5",
    author="Сидоров А.В.",
    date="2025-11-04"
)
```
