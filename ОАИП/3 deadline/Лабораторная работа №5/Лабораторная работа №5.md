#### Задача 1 (Класс и атрибуты)
Создайте класс `Student`, который при инициализации (метод `__init__`) принимает:
*   `name` (имя студента)
*   `average_score` (средний балл)

Добавьте в класс метод `is_excellent()`, который возвращает `True`, если средний балл 5, и `False` в противном случае.

**Пример использования:**
```python
student1 = Student("Vasya", 4.5)
student2 = Student("Petya", 5)

print(student1.is_excellent())
print(student2.is_excellent())
```
**Вывод:**
```
False
True
```

```python
class Student:
    def __init__(self, name, average_score):
        self.name = name
        self.average_score = average_score
    def is_excellent(self):
        if self.average_score == 5:
            return True
        else:
            return False
student1 = Student("Vasya", 4.5)  
student2 = Student("Petya", 5)  
print(student1.is_excellent())  
print(student2.is_excellent())
```
#### Задача 2 (Методы и изменение состояния)
Реализуйте класс `BankAccount` (Банковский счет).
Атрибуты:
*   `account_holder` (владелец счета)
*   `balance` (баланс, по умолчанию 0)

Методы:
1.  `deposit(amount)`: добавляет `amount` к балансу.
2.  `withdraw(amount)`: снимает `amount` с баланса. Если на счету недостаточно средств, выводит сообщение об ошибке и не меняет баланс.
3.  `get_balance()`: возвращает текущий баланс.

**Пример использования:**
```python
account = BankAccount("Ivanov", 100)
account.deposit(50)
account.withdraw(200) # Ошибка!
account.withdraw(30)
print(account.get_balance())
```
**Вывод:**
```
Недостаточно средств!
120
```

```python
class BankAccount:
    def __init__(self, account_holder, balance=0):
        self.account_holder = account_holder
        self.balance = balance
    def deposit(self, amount):
        self.balance = self.balance + amount
    def withdraw(self, amount):
        if amount > self.balance:
            print("Недостаточно средств!")
        else:
            self.balance = self.balance - amount
    def get_balance(self):
        return self.balance
account = BankAccount("Ivanov", 100)  
account.deposit(50)  
account.withdraw(200) 
account.withdraw(30)  
print(account.get_balance())
```
#### Задача 3 (Наследование)
Создайте базовый класс `Animal`:
*   Атрибуты: `name`.
*   Метод `make_sound()`: выводит "Животное издает звук".

Создайте два дочерних класса:
1.  `Dog` (Собака). Переопределите метод `make_sound()`, чтобы он выводил "Гав!".
2.  `Cat` (Кошка). Переопределите метод `make_sound()`, чтобы он выводил "Мяу!".

Напишите функцию (вне классов) `animal_chorus(animals)`, которая принимает список объектов животных и вызывает у каждого метод `make_sound()`.

**Пример использования:**
```python
animals = [Dog("Sharik"), Cat("Matroskin")]
animal_chorus(animals)
```
**Вывод:**
```
Гав!
Мяу!
```

```python
class Animal:
    def __init__(self, name):
        self.name = name
    def make_sound(self):
        print("Животное издает звук")
class Dog(Animal):
    def make_sound(self):
        print("Гав!")
class Cat(Animal):
    def make_sound(self):
        print("Мяу!")
def animal_chorus(animals):
    for animal in animals:
        animal.make_sound()
animals = [Dog("Sharik"), Cat("Matroskin")]
animal_chorus(animals)
```
#### Задача 4 (Взаимодействие объектов)
Это классическая задача на ООП. Реализуйте простую боевую систему.

1.  Класс `Hero`:
    *   Атрибуты: `name`, `health` (здоровье, по умолчанию 100), `attack_power` (сила удара, по умолчанию 20).
    *   Метод `strike(target)`: атакует другого героя (`target`). Отнимает у `target` количество здоровья, равное `attack_power` атакующего.

2.  В коде создайте двух героев и заставьте их ударить друг друга. Выведите здоровье обоих после драки.

**Важно:** Метод `strike` должен принимать именно **объект** другого героя, а не просто число.

**Пример использования:**
```python
knight = Hero("Arthur", 100, 20)
rogue = Hero("Robin", 80, 15)

knight.strike(rogue) # Артур бьет Робина
rogue.strike(knight) # Робин бьет Артура

print(f"{knight.name}: {knight.health}")
print(f"{rogue.name}: {rogue.health}")
```
**Вывод:**
```
Arthur: 85
Robin: 60
```

```python
class Hero:
    def __init__(self, name, health=100, attack_power=20):
        self.name = name
        self.health = health
        self.attack_power = attack_power
    def strike(self, target):
        target.health = target.health - self.attack_power
knight = Hero("Arthur", 100, 20)
rogue = Hero("Robin", 80, 15)
knight.strike(rogue) 
rogue.strike(knight) 
print(f"{knight.name}: {knight.health}")
print(f"{rogue.name}: {rogue.health}")
```
#### Задача 5 (Список объектов)
Создайте класс `Task` (Задача) с атрибутами:
*   `description` (описание задачи)
*   `priority` (приоритет, целое число)

Создайте класс `TaskManager` (Менеджер задач), который содержит список задач (изначально пустой).
Методы `TaskManager`:
1.  `add_task(description, priority)`: создает новую задачу и добавляет в список.
2.  `show_tasks()`: выводит все задачи в формате "Описание - Приоритет".

**Пример использования:**
```python
manager = TaskManager()
manager.add_task("Купить хлеб", 1)
manager.add_task("Сделать домашку", 10)
manager.show_tasks()
```
**Вывод:**
```
Купить хлеб - 1
Сделать домашку - 10
```

```python
class Task:
    def __init__(self, description, priority):
        self.description = description
        self.priority = priority
class TaskManager:
    def __init__(self):
        self.tasks = [] 
    def add_task(self, description, priority):
        new_task = Task(description, priority)
        self.tasks.append(new_task)
    def show_tasks(self):
        for task in self.tasks:
            print(f"{task.description} - {task.priority}")
manager = TaskManager()
manager.add_task("Купить хлеб", 1)
manager.add_task("Сделать домашку", 10)
manager.show_tasks()
```
